# Helix

**Predictive Biomedical & Public Health Intelligence Platform**

Team: SPECTRANET (JSS Science and Technology University)

## Overview
Helix is an advanced, AI-driven platform for tracking, predicting, and visualizing public health trends and outbreaks globally.

## Prerequisites
- Node.js & npm
- Python 3.10+
- Git

## How to Run

1. **Install dependencies:**
   - **Frontend**: `cd frontend && npm install`
   - **Backend**: `cd backend && pip install -r requirements.txt`
   - **ML Service**: `cd ml && pip install -r requirements.txt`

2. **Start the application:**
   For Mac/Linux:
   ```bash
   ./start.sh
   ```
   For Windows:
   ```powershell
   .\start.ps1
   ```

3. **Access the services:**
   - Frontend (React): http://localhost:5173
   - Backend (FastAPI): http://localhost:8000
   - ML Service (FastAPI): http://localhost:8001
